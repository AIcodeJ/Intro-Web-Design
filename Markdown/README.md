# Markdown Style #

<!-- UL -->
* Item 1
    * subItem 1
<!-- OL -->
1. First Item
    1. subItem 1

<!-- italic -->
*this is an italic text*

<!-- bold -->
**this is an strong text**

<!-- crossed text -->
~~this is a crossed text~~

<!-- links -->
[Github](https://github.com) <--- Link for the Github page

<!-- Quote -->
>This is quote

<!-- Lines -->
---
___

<!-- Inline code -->
`console.log("Hello World")`

_or_

```Javascript
console.log("Hello World")
```
